记个人博客诞生

发表于 2019-01-09 | 分类于 学习

历程

  之前从来没搭建过博客，所以这次是零经验开始，边摸索，边应用。主要手段是百度查阅资料，动手之前就找到一份详细教程，所以总体来说没有遇到什么大问题。感谢作者，教程地址：hexo个人博客搭建教程。捉个虫，菜单的icon并不是在menu_icon下面设置的，而是在每一条menu后面加上“|| iconname”。次要手段是求助Cy大神，感谢Cy，大神的github：Young。总共历时两天，顺利搭建第一个个人博客！

工具&技术

  • Github&GitBash
  • Nodejs
  • Hexo
  • NexT
  • Live2d
  • 域名管理

收获

  复习了一下Git的用法和一些基础命令，其他的技术和工具都是第一次使用。node用的不多，只是使用npm命令安装Hexo和一些插件。
  Hexo是博客框架，通过命令行来操作，基于这个框架，只需要写好MarkDown文件就可以使用命令直接发布。可以修改_config.yml文件来定制一些个性化的内容。
  NexT是博客的主题插件，预设了四种主题，都很简洁清新，并且提供了国际化、菜单栏、头像框等一些很实用的功能。同样有一个_config.yml文件，用来定制个性化内容。
  Live2d看板娘，使用git从github上拉取下来的插件，可以监听鼠标位置以及部分鼠标事件。头部会自动跟随鼠标位置进行转向，也可以定制一些由鼠标事件触发的其他动作。
  域名管理是第一次接触，解析域名的时候填好个人博客的IP地址，然后在Github的博客项目上绑定好域名即可。期间Github提示建议使用CNAME记录，所以了解了一下CNAME记录和A记录的区别，A记录直接解析IP，而CNAME记录解析主机名。相关材料：域名解析各种记录类型的区别。
